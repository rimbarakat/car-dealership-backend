# car-dealership-backend

## Signup and Login:
### Prerequisites:
Install all the necessary packages in the package-json file.   
MongoDB connection.   
For the timebeing you need to have postman.   

### End-Points:
http://localhost:Portnumber/auth/register         
http://localhost:Portnumber/auth/login         

### Technologies Used:
Node.js     
Express     
MongoDB     
JWT     
Crypto-js
